📸 RSCapture - Photography Portfolio Website

A modern and responsive photography portfolio website designed to showcase stunning visual stories, 
photography services, and creative work. Built with a clean user interface and elegant design, RSCapture 
helps photographers present their portfolio professionally and engagingly.

🌐 Live Demo
https://anand-chauhan.github.io/rscapture/

✨ Features
Modern and Responsive Design
Beautiful Hero Section
Photography Portfolio Gallery
Contact Section
Smooth Scrolling Navigation
Mobile-Friendly Layout
Cross-Browser Compatibility
Fast Loading Performance

🛠️ Technologies Used
HTML5
CSS3
JavaScript
Bootstrap
Font Awesome

🎯 Purpose

This project was created to provide photographers and creative professionals with a 
visually appealing platform to showcase their work, attract potential clients, and build an online presence.
